# Необходимые ресурсы

## 1. Общая информация

**Название проекта:** ИИNEUROCHIEF × Фабрика здорового питания  
**Дата создания:** 2025-01-27  
**Версия документа:** 1.0  
**Период:** Полный цикл разработки (MVP + дальнейшее развитие)

**Особенности проекта:**
- Промышленная ИИ-система с компьютерным зрением
- Требуются специалисты по ML/CV
- Интеграция с производственным оборудованием

## 5. Необходимые сервисы / подписки / инфраструктура

### 5.1. Облачная инфраструктура

#### Основная платформа (выбор одной)
- **AWS** (Amazon Web Services)
  - EC2 (для сервисов), RDS (PostgreSQL), S3 (видео/изображения), CloudFront
  - SageMaker (для обучения ML моделей, опционально)
  - Примерная стоимость: $800-2500/месяц (зависит от нагрузки)
  
- **Google Cloud Platform (GCP)**
  - Compute Engine, Cloud SQL, Cloud Storage
  - Vertex AI (для ML, опционально)
  - Примерная стоимость: $800-2500/месяц
  
- **Microsoft Azure**
  - Virtual Machines, Azure SQL, Blob Storage
  - Azure ML (опционально)
  - Примерная стоимость: $800-2500/месяц

**Рекомендация:** Google Cloud (хорошая поддержка ML) или AWS

#### Edge Infrastructure (Периферийные устройства)

  - Jetson AGX Orin / Jetson Xavier NX
  - Примерная стоимость: $2000-4000 за устройство
  - Количество: 2-4 устройства для пилота
  
- **Промышленные ПК** (альтернатива Jetson)
  - Примерная стоимость: $1500-3000 за устройство
  - Количество: 2-4 устройства для пилота

- **Камеры** (если не предоставлены заказчиком)
  - IP камеры с RTSP поддержкой
  - Примерная стоимость: $200-500 за камеру

**Итого для edge инфраструктуры:** $10,000-25,000 (единоразово для пилота)
